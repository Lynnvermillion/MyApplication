User stories

General User
As a user, I want to check the weather for the next day to check the app suggestions for what to wear.
As a user, I want to be able to store my clothes, so the app knows what to suggest.
As a user, I want to be able to store my outfit so I can pick an outfit that I like manually.
As a user I want to remove any clothes/outfits that I no longer have or that I do not like, the app should not recommend me the removed clothes/outfits.
As a user, I want to be able to store my preferences so that the app will know better what to suggest.
As a user I want to be able to select the occasion so the app can recommend appropriate clothes.
As a user, I want to be able to categorize my clothing into casual, posh etc. so the app can recommend outfits based on the occasion. 
As a user I want to share my suggested outfit so my friends can see it.
As a user I want to get notifications of the weather daily so it can be aware all the time.
As a user I should be able to select specific weather and it should show me couple destinations with that weather.
